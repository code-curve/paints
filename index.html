    Paints

    Paints is a nodejs color scheme project designed to give js hackers and user interface designers quick access to an arbitrary number of colors based on one of the many themes here. Check out some examples.

    Get paints for your browser
    Get paints.js or paints.min.js

    Get paints for nodejs
    npm install paints

    Contribute
    Got an idea for a theme? Pop over to the GitHub page to get involved!
